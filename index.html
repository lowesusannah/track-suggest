Which programming language should I learn first?

Chosing your first programming language can be a difficult choice! After all, you are probably not familiar with all of the options out there, or how any language works. But that's OK -- using our handy quiz below, find out which language is likely to be the best one for you to start out in at Epicodus. From there, you're certain to understand other languages better and be less intimidated to learn other languages.
